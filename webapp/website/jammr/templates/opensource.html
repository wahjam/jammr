Open source

How we contribute to open source and you can too

We want you to have the freedom to learn, tinker, and extend!

jammr uses the open source development model to create and grow our online jamming platform.

jammr is the driving force behind Wahjam, the open source online jamming software that powers the jammr client and session server. All our code for the client and session server is contributed to the community.

Both the client and the session server are under the GNU Public License v2 or Later.

$ git clone https://github.com/wahjam/wahjam.git

The jammr website and backend infrastructure is not open source at this time.

For discussions about the client and session server code, join the gitter.im chat or connect to #wahjam_community:gitter.im using your Matrix chat client.

Join the community!

Meet us on the forum. We love to hear your ideas, help you when you're stuck, and build a great jamming community!
